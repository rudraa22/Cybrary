## 📚 Learning Resources

### Free Platforms
- [TryHackMe](https://tryhackme.com) — Beginner-friendly, guided rooms
- [Hack The Box](https://hackthebox.com) — Intermediate to advanced labs
- [PortSwigger Web Academy](https://portswigger.net/web-security) — Best web security course, free
- [OWASP](https://owasp.org) — Gold standard for web security knowledge
- [SANS Reading Room](https://www.sans.org/white-papers/) — In-depth whitepapers

### YouTube Channels
- **John Hammond** — CTF walkthroughs & malware analysis
- **NetworkChuck** — Beginner-friendly networking & hacking
- **IppSec** — HTB machine walkthroughs (no-skip material)
- **David Bombal** — Networking, Python, ethical hacking

### Certifications Roadmap
```
Beginner   →  CompTIA Security+
Intermediate → CEH · eJPT · OSCP
Advanced   →  CISSP · OSWE · CRTE
```
